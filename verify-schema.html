Method 1: View Page Source (Best Method)

  1. Open your homepage: http://localhost:3000
  2. Right-click anywhere on the page
  3. Select "View Page Source" (or press Ctrl+U / Cmd+Option+U)
  4. Press Ctrl+F (or Cmd+F on Mac) to open search
  5. Search for: application/ld+json
  6. You should see exactly ONE script tag

Method 2: Browser Console (Quick Check)

  1. Open your homepage: http://localhost:3000
  2. Press F12 to open DevTools
  3. Go to "Console" tab
  4. Paste and run this code:
document.querySelectorAll('script[type="application/ld+json"]').forEach((script, index) => { console.log(`Schema ${index + 1}:`, JSON.parse(script.textContent)); }); console.log(`Total schemas found: ${document.querySelectorAll('script[type="application/ld+json"]').length}`);

Expected Result: Total schemas found: 1

Method 3: What to Look For

In the page source, you should find this structure:

<script type="application/ld+json"> { "@context": "https://schema.org", "@graph": [ { "@type": "Organization", "@id": "https://www.techpotlidigital.com/#organization", "name": "Techpotli Digital", ... }, { "@type": "WebSite", "@id": "https://www.techpotlidigital.com/#website", ... } ] } </script>

‚úÖ Check for:

  • Only ONE script tag with type="application/ld+json"
  • @graph array present
  • Both Organization and WebSite types
  • Proper @id references

  3. Check for any errors
